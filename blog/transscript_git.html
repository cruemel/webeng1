Transkript Git

Konfiguration

  • git config --global user.name "Vorname Nachname"
  • git config --global user.email "vorname.nachname@mailprovider.de"
  • git config --global color.ui auto

Lokale Operationen

Initiierung eines Verzichnisses für Git
git init
Der aktuelle Stand
git status
Datei zur Überwachung hinzufügen
git add [DATEI]
Mehrere Dateien hinzufügen
git add *
Bestätigung (und Kommentierung) der Änderungen
git commit -m "[NACHRICHT]"
Datei löschen
git rm [DATEI]

Entfernte Operationen

Repository kopieren
git clone [URL]
Repository aktualisieren
git pull
Ein Repository mit Github verbinden
git remote add origin https://github.com/[BENUTZER]/[REPRO]
Dateien an Github senden
git push [REMOTE] [BRANCH]

Änderungen

Datei(en) vergleichen
git diff (HEAD) [DATEI]
Änderungen herunterladen
git fetch [DATEI]
Änderungen zusammenführen
git merge [BRANCH]
Änderungen an einer Datei zurücknehmen
git checkout -- [DATEI]
Herausnehmen einer Datei aus dem Repository
git reset HEAD [DATEI]

Publizieren

  1. git status - was hat sich geändert?
  2. git add [DATEI] - veränderte - und neue - Dateien zum Index hinzuzufügen.
  3. git commit -m "[NACHRICHT]" - die Änderungen kommentieren
  4. git push origin master - Änderungen an Github senden
