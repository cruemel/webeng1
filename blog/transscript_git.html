Transkript Git

Konfiguration

  • git config --global user.name "Vorname Nachname"
  • git config --global user.email "vorname.nachname@mailprovider.de"
  • git config --global color.ui auto

Initiialisierung

Neues Verzichnisses für Git anlegen
git init
Lokales Repository kopieren
git clone [PATH]
Entferntes Repository kopieren
git clone [USER]@[URL]

Dateien hinzufügen (oder löschen)

Datei zur Überwachung hinzufügen
git add [DATEI]
Mehrere Dateien hinzufügen
git add *
Datei löschen
git rm [DATEI]

Bestätigen und synchronisieren

Bestätigung (und Kommentierung) der Änderungen
git commit -m "[NACHRICHT]"
Änderungen an das entfernte Repository senden
git push origin master
Lokales Repository mit einem entfernten verbinden
git remote add origin [SERVER]/[BENUTZER]/[REPRO]
Ein lokales Repository aktualisieren
git pull

Zweige

Einen neuen Zweig anlegen
git checkout -b [BRANCH]
Wechseln zum Master-Zweig
git checkout master
Einen Zweig löschen
git branch -d [BRANCH]
Einen Zweig an das entfernte Repository senden
git push origin [BRANCH]

Zusamenfügen

Änderungen zusammenführen
git merge [BRANCH]
Zwei Zweige vergleichen
git diff [BRANCH_1] [BRANCH_2]

Publizieren

  1. git status - was hat sich geändert?
  2. git add [DATEI] - veränderte - und neue - Dateien zum Index hinzuzufügen.
  3. git commit -m "[NACHRICHT]" - die Änderungen kommentieren
  4. git push origin master - Änderungen an Github senden
