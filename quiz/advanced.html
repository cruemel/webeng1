Was ist CI? CI steht für Continuous Integration und beschreibt den Mechanismus der Automatisierung von Integrationprozessen

Welche Vorteile hat CI? die Qualität des Endprodukts wird verbessert. ein Build kann jederzeit ausgeliefert werden. der Build Prozess wird dokumentiert. die Wahrscheinlichkeit von Problemen bei Projektende oder einem Meilenstein wird geringer

Welche Aufgaben hat ein Scrum Master? Er sorgt dafür, dass die Scrum-Regeln von allen Beteiligten eingehalten werden. Er moderiert die Scrum-Meetings und kümmert sich um die Behebung von Problemen im Entwicklerteam. Er kümmert sich darum, dass die Scrum-Artefakte (Backlogs, Burndown Chart usw.) stets aktuell gehalten werden.

Was ist ein Burnout Chart? Eines der Artefakte im Scrum Prozess. Er visualisiert die voraussichtlich noch zu leistende Arbeit, bis das gesetzte Ziel erreicht wird. Mittels einer, auf mehreren Schätzwerten basierenden Trendlinie kann das voraussichtliche zeitliche Ende vorhergesagt werden

Für einen effektiven Einsatz von "Continuous Integration" hat Martin Fowler einige Grundsätze aufgestellt. Maintain a Single Source Repository. - Automate the Build - Make Your Build Self-Testing - Everyone Commits To the Mainline Every Day - Every Commit Should Build the Mainline on an Integration Machine - Keep the Build Fast - Test in a Clone of the Production Environment - Make it Easy for Anyone to Get the Latest Executable - Everyone can see what's happening - Automate Deploymen

Erläutern Sie den Unterschied zwischen den Befehlen $vagrant destroy, $vagrant halt und $vagrant suspend. $vagrant destroy: Dieser Befehl löscht die aktuelle VM und alle während der Laufzeit angelegten Ressourcen. $vagrant halt: Mit diesem Befehl wird die aktuelle VM heruntergefahren. $vagrant suspend: Speichert den aktuellen Zustand der VM und versetzt diese beim nächsten Start wieder in den selbigen

Welche Aufgabe erfüllen Assertions in JMeter? Um die ausgeführten Tests auch inhaltlich überprüfen zu können, werden Assertions eingesetzt. Diese können beispielsweise mit Hilfe von regulären Ausdrücken die gelieferten Ergebnisse überprüfen oder die Antwortzeit mit einem festgelegten Schwellwert vergleichen

Welche Bedeutung haben die folgenden Zeilen in einem Vagrant File? config.vm.box = "webdev" config.vm.box_url = "http://files.vagrantup.com/precise32.box config.vm.box gibt den Namen der zu verwendenden Box an. Sollte die entsprechende Box noch nicht auf dem Rechner verfügbar sein, so gibt config.vm.box_url den Download Link an, an der die entsprechende Box hinterlegt ist und lädt diese automatisch herunter.
