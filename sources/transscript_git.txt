# Die wichtigsten Kommandos zu Git

1. Konfiguration
    
    git config --global user.name "Vorname Nachname"
    git config --global user.email "vorname.nachname@mailprovider.de"
    git config --global color.ui auto
   
2. Verzeichnisoperationen

    Abfrage des Git-Status
    git status

    Initiierung des Verzichnisses fuer Git
    git init

    Datei zur Ueberwachung hinzufuegen
    git add [DATEI]
    
    Mehrere Dateien hinzufuegen
    git add *
    
    Herausnehmen einer Datei aus dem Repository
    git reset HEAD [DATEI]

    Zuruecknehmen von Aenderungen
    git checkout -- [DATEI]
    
    Datei loeschen
    git rm [DATEI]

    Bestaetigung der Aenderungen
    git commit -m "[NACHRICHT]"
    
3. Publizieren

	1. git status - welche Dateien haben sich geaendert?
	2. git add [DATEI] - Datei(en) hinzuzufuegen.
	3. git commit -m "[NACHRICHT]" - Aenderungen kommentieren
	4. git push origin master - Aenderungen an Github senden
