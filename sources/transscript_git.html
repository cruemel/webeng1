Die wichtigsten Kommandos zu Git

1. Konfiguration

  • git config --global user.name "Vorname Nachname"
  • git config --global user.email "vorname.nachname@mailprovider.de"
  • git config --global color.ui auto

2. Verzeichnisoperationen

Abfrage des Git-Status
git status
Initiierung des Verzichnisses für Git
git init
Datei zur Überwachung hinzufügen
git add [DATEI]
Mehrere Dateien hinzufügen
git add *
Herausnehmen einer Datei aus dem Repository
git reset HEAD [DATEI]
Zurücknehmen von Änderungen
git checkout -- [DATEI]
Datei löschen
git rm [DATEI]
Bestätigung der Änderungen
git commit -m "[NACHRICHT]"

3. Publizieren

  1. git status - welche Dateien haben sich geändert?
  2. git add [DATEI] - Datei(en) hinzuzufügen.
  3. git commit -m "[NACHRICHT]" - Änderungen kommentieren
  4. git push origin master - Änderungen an Github senden
